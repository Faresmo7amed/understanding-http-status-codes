# Understanding HTTP Status Codes
A beginner-friendly HTML documentation page explaining HTTP status codes, their meanings, and how they are used in REST APIs.

## Overview

This project is a semantic HTML article that explains the five categories of HTTP status codes:

- 1xx - Informational Responses
- 2xx - Successful Responses
- 3xx - Redirection Responses
- 4xx - Client Error Responses
- 5xx - Server Error Responses

It also includes:

- HTTP status code tables
- RFC 9110 references
- Request/response examples
- Glossary of HTTP terms
- Footnotes and citations
- HTML forms and semantic elements

## Technologies Used
